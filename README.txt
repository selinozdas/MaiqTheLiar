CS465 2. Assignment:

This is a 3d cat with animations.

How to use:

Change the angles of joints by using sliders. After playing with sliders, you can save multiple frames, then animate it.
You can also save a frame to your file system and load a animation from file system.
You can use predefined walk and run animations.


Common files:

webgl-utils.js: standard utilities from google to set up a webgl context

MV.js: our matrix/vector package. Documentation on website

initShaders.js: functions to initialize shaders in the html file
